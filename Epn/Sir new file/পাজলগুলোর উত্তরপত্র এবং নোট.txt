# পাজলগুলোর উত্তরপত্র এবং নোট (Notes of Ideal Answer Sheet to TACM)
Answer Sheet to Puzz 6.

---

## পাজল ৬: ফাংশনাল ইকুয়েশন (Functional Equation)

**সমস্যা:** এমন সকল ফাংশন f : ℝ → ℝ নির্ণয় করুন যেন সকল বাস্তব সংখ্যা x, y এর জন্য:
f(x·f(y) + f(x)) = 2f(x) + xy ... (1) হয়।

---

### পর্ব ১: দাবিকৃত সমাধানের যাচাই (Verification)

অনন্যতা (uniqueness) প্রমাণের আগে, আসুন যাচাই করে দেখি যে f(x) = x + 1 কাজ করে কি না।

**বামপক্ষ (LHS):**
f(x·f(y) + f(x)) = f(x(y+1) + (x+1)) = f(xy + x + x + 1) = f(xy + 2x + 1)
= (xy + 2x + 1) + 1 = xy + 2x + 2

**ডানপক্ষ (RHS):**
2f(x) + xy = 2(x+1) + xy = 2x + 2 + xy

সকল বাস্তব সংখ্যা x, y এর জন্য, বামপক্ষ = ডানপক্ষ = xy + 2x + 2। ✅

সুতরাং f(x) = x + 1 নিশ্চিতভাবে একটি সমাধান।
এখন প্রশ্ন হলো: এটিই কি *একমাত্র* সমাধান? চলুন প্রমাণ করা যাক।

---

### পর্ব ২: f(x) = x + 1 যে অনন্য সমাধান তা প্রমাণ করা

---

#### ধাপ ১: f(0) নির্ণয়
সমীকরণ (1)-এ x = 0 বসাই:
f(0 · f(y) + f(0)) = 2f(0) + 0 · y
f(f(0)) = 2f(0) ... (2)

এটি আমাদের f(0) সম্পর্কে কিছু ধারণা দেয়, কিন্তু সঠিক মানটি এখনও জানা যায়নি। এটি মনে রাখুন।

#### ধাপ ২: প্রমাণ করা যে f হলো ইনজেক্টিভ (injective বা এক-এক)
ধরি, কোনো বাস্তব সংখ্যা a, b এর জন্য f(a) = f(b)।

মূল সমীকরণে x = 1 বসাই:
f(1 · f(y) + f(1)) = 2f(1) + 1 · y
f(f(y) + f(1)) = 2f(1) + y ... (3)

এখন (3) নং সমীকরণে y এর জায়গায় পর্যায়ক্রমে a এবং b বসাই:
f(f(a) + f(1)) = 2f(1) + a
f(f(b) + f(1)) = 2f(1) + b

যেহেতু f(a) = f(b), তাই বামপক্ষ দুটি অভিন্ন। অতএব,
2f(1) + a = 2f(1) + b ⇒ a = b

সুতরাং, f একটি **ইনজেক্টিভ** ফাংশন।

#### ধাপ ৩: প্রমাণ করা যে f হলো সারজেক্টিভ (surjective বা সার্বিক)
(1) নং সমীকরণে x = 1 বসাই:
f(f(y) + f(1)) = 2f(1) + y ... (4)

এখন y কে সকল বাস্তব সংখ্যার উপর পরিবর্তিত হতে দিই। যেহেতু y পরিবর্তিত হয়, ডানপক্ষ *প্রতিটি বাস্তব মান* গ্রহণ করে (কারণ 2f(1) + y হলো y এর একটি রৈখিক ফাংশন যার ঢাল 1 — যা ℝ থেকে ℝ-এ একটি বাইজেকশন)।

সুতরাং, g(y) = f(f(y) + f(1)) ফাংশনটি সারজেক্টিভ। যেহেতু f ∘ h সারজেক্টিভ হলে f সারজেক্টিভ হতে বাধ্য, আমরা সিদ্ধান্তে আসতে পারি যে f একটি **সারজেক্টিভ** ফাংশন।

অতএব f হলো **বাইজেক্টিভ** (bijective)।

#### ধাপ ৪: নিখুঁতভাবে f(0) নির্ণয়
যেহেতু f সারজেক্টিভ, তাই এমন কোনো বাস্তব সংখ্যা t আছে যার জন্য f(t) = 0 হয়।

মূল সমীকরণ (1)-এ y = t বসাই:
f(x · f(t) + f(x)) = 2f(x) + xt
যেহেতু f(t) = 0:
f(0 + f(x)) = 2f(x) + xt
f(f(x)) = 2f(x) + xt ... (5)

এখন (5) নং এ x = t বসাই:
f(f(t)) = 2f(t) + t · t
যেহেতু f(t) = 0:
f(0) = 0 + t^2 = t^2 ... (6)

(2) নং থেকে আমরা পাই f(f(0)) = 2f(0)। f(0) = t^2 বসালে:
f(t^2) = 2t^2 ... (7)

#### ধাপ ৫: সবকিছু যুক্ত করতে একটি চমৎকার প্রতিস্থাপন
মূল সমীকরণ (1)-এ x এবং y এর স্থান অদলবদল (swap) করি:
f(y·f(x) + f(y)) = 2f(y) + yx ... (8)

ডানপক্ষটি প্রতিসম (symmetric): 2f(y) + yx = xy + 2f(y)। মূল সমীকরণের ডানপক্ষ ছিল 2f(x) + xy।
তাই:
f(x·f(y) + f(x)) = f(y·f(x) + f(y)) ... (9)

যেহেতু f **ইনজেক্টিভ**, তাই এর আর্গুমেন্টগুলো সমান হতে হবে:
x·f(y) + f(x) = y·f(x) + f(y) ... (10)

এটি একটি শক্তিশালী ফাংশনাল সম্পর্ক। পুনর্বিন্যাস করলে:
x·f(y) - f(y) = y·f(x) - f(x)
f(y)(x - 1) = f(x)(y - 1) ... (11)

#### ধাপ ৬: রৈখিক রূপ (linear form) বের করা
সমীকরণ (11) **সকল বাস্তব সংখ্যা x, y** এর জন্য সত্য।

x ≠ 1 এবং y ≠ 1 এর জন্য আমরা লিখতে পারি:
f(x) / (x - 1) = f(y) / (y - 1)

এর মানে হলো সকল x ≠ 1 এর জন্য f(x) / (x - 1) অভিব্যক্তিটি **ধ্রুবক (constant)**। ধরি সেই ধ্রুবকটি হলো c। তাহলে:
f(x) = c(x - 1)  (সকল x ≠ 1 এর জন্য)

সারজেক্টিভিটি এবং সমাধান রূপের অবিচ্ছিন্নতার (continuity) কারণে (আমরা লিমিট নিতে পারি অথবা f(1) নির্ধারণের জন্য x = 1 এ ফাংশনাল সমীকরণ ব্যবহার করতে পারি), এই সূত্রটি x = 1 এর জন্যও কাজ করবে।

#### ধাপ ৭: ধ্রুবক c নির্ধারণ
আমরা জানি কোনো t এর জন্য f(t) = 0। f(x) = c(x - 1) ব্যবহার করে:
f(t) = c(t - 1) = 0
সুতরাং হয় c = 0 অথবা t = 1।

**ক্ষেত্র ১: c = 0।** তাহলে সকল x এর জন্য f(x) = 0। মূল সমীকরণে বসালে: f(x·f(y)+f(x)) = f(0) = 0। ডানপক্ষ: 2f(x) + xy = 0 + xy = xy। সকল x, y এর জন্য, 0 = xy — যা মিথ্যা। তাই c ≠ 0।

**অতএব, t = 1।** তাহলে f(1) = c(1-1) = 0।

সুতরাং f(1) = 0, এবং f(t) = 0 যেখানে t = 1, যা সামঞ্জস্যপূর্ণ।

#### ধাপ ৮: মূল সমীকরণ থেকে c বের করা
এখন আমাদের কাছে আছে f(x) = c(x - 1), এবং আমরা জানি f(1) = 0।
মূল সমীকরণ (1)-এ f(x) = c(x-1) বসাই:

**বামপক্ষ (LHS):**
প্রথমে হিসাব করি f(y) = c(y-1), তারপর x·f(y) = xc(y-1)। আবার f(x) = c(x-1)।
আর্গুমেন্ট = x·f(y) + f(x) = cx(y-1) + c(x-1) = c[xy - x + x - 1] = c(xy - 1)
তাহলে:
বামপক্ষ = f(c(xy - 1)) = c(c(xy - 1) - 1) = c^2·xy - c^2 - c

**ডানপক্ষ (RHS):**
2f(x) + xy = 2c(x-1) + xy = xy + 2cx - 2c

**সমীকরণ মেলানো:**
c^2·xy - c^2 - c = xy + 2cx - 2c

এটি **সকল বাস্তব সংখ্যা x, y** এর জন্য সত্য হতে হবে।

সহগ (coefficients) তুলনা করে:
- xy এর সহগ: c^2 = 1 ⇒ c = 1 অথবা c = -1।
- x এর সহগ: বামপক্ষে কোনো স্বাধীন x টার্ম নেই (0)। ডানপক্ষে: 2c। সুতরাং 0 = 2c ⇒ c = 0।

কিন্তু আমরা আগেই c = 0 বাতিল করেছি, এবং xy সহগ থেকে আমরা পেয়েছি c = ± 1। x টার্ম থেকে আসা c = 0 এর সাথে এটি কি সাংঘর্ষিক?

দাঁড়ান — চলুন এটি সতর্কতার সাথে আবার চেক করি।

**বামপক্ষ (LHS) সম্প্রসারণের পুনরায় চেক:**
আর্গুমেন্ট = x·f(y) + f(x) = x · c(y-1) + c(x-1) = cxy - cx + cx - c = cxy - c।

আহ! -cx এবং +cx কাটাকাটি যায়! তাই:
আর্গুমেন্ট = cxy - c
তাহলে:
বামপক্ষ = f(cxy - c) = c((cxy - c) - 1) = c(cxy - c - 1) = c^2·xy - c^2 - c

**ডানপক্ষ (RHS):**
2f(x) + xy = 2c(x-1) + xy = xy + 2cx - 2c

**সমীকরণ:**
c^2·xy - c^2 - c = 1 · xy + 2c · x - 2c

এটি **সকল x, y** এর জন্য সত্য হতে হলে:
1. xy এর সহগ: c^2 = 1 ⇒ c = 1 অথবা c = -1।
2. x এর সহগ: বামপক্ষে = 0, ডানপক্ষে = 2c। তাই 0 = 2c ⇒ c = 0।

এটি একটি সাংঘর্ষিক অবস্থা (Contradiction) তৈরি করে... ডানপক্ষের x এর সহগ আরেকবার পরীক্ষা করি।

ডানপক্ষটি হলো xy + 2cx - 2c। যখন আমরা "x এর সহগ" বলি, তখন y-কে একটি স্বাধীন চলক হিসেবে ধরা হয়। পরীক্ষা করতে নির্দিষ্ট মান বসাই:

y = 0 বসাই। সমীকরণটি দাঁড়ায়:
c^2 · x · 0 - c^2 - c = x · 0 + 2cx - 2c
-c^2 - c = 2cx - 2c
এটি সকল x এর জন্য সত্য হতে হবে। বামপক্ষ ধ্রুবক; ডানপক্ষ x-এ রৈখিক যদি না 2c = 0 হয়। তাই c = 0।

কিন্তু c = 0, xy টার্মের সাথে সাংঘর্ষিক যেখানে c^2 = 1 প্রয়োজন।

**এটি ইঙ্গিত দিচ্ছে যে f(x) = c(x-1) এই রূপটি বের করায় আমাদের ভুল ছিল, অথবা সূক্ষ্ম কোনো বিষয় বাদ পড়েছে।**

#### ধাপ ৯: সমীকরণ (11) পুনরায় পরীক্ষা
আমাদের ছিল:
f(y)(x - 1) = f(x)(y - 1)   (সকল x, y ∈ ℝ এর জন্য)

y = 1 বসাই:
f(1)(x - 1) = f(x) · 0
f(1)(x - 1) = 0   (সকল x এর জন্য)

এটি f(1) = 0 হতে বাধ্য করে। ঠিক আছে।

এখন x = 0 বসাই:
f(y)(0 - 1) = f(0)(y - 1)
-f(y) = f(0)(y - 1)
f(y) = -f(0)(y - 1) = f(0)(1 - y)

ধরি c = -f(0), তাহলে সকল y এর জন্য f(y) = c(y - 1)।

সুতরাং f(x) = c(x-1) আকারটি সঠিক, যেখানে c = -f(0)।

#### ধাপ ১০: সাংঘর্ষিক অবস্থার সমাধান (Resolve the contradiction)
যাচাই করার সময় আমি নিশ্চয়ই অ্যালজেবরায় কোনো ভুল করেছি। মূল সমীকরণে চরম সতর্কতার সাথে f(x) = c(x-1) আবার বসাই।

**প্রদত্ত:** সকল x এর জন্য f(x) = c(x-1), যেখানে c একটি ধ্রুবক।

**x·f(y) হিসাব করি:**
f(y) = c(y-1), তাই x·f(y) = cx(y-1) = cxy - cx।

**x·f(y) + f(x) হিসাব করি:**
= (cxy - cx) + c(x-1) = cxy - cx + cx - c = cxy - c।

তাই আর্গুমেন্ট হলো cxy - c = c(xy - 1)।

**বামপক্ষ = f(cxy - c) হিসাব করি:**
= c((cxy - c) - 1) = c(cxy - c - 1) = c^2·xy - c^2 - c। ✅

**ডানপক্ষ = 2f(x) + xy হিসাব করি:**
f(x) = c(x-1) = cx - c
2f(x) = 2cx - 2c
2f(x) + xy = xy + 2cx - 2c। ✅

**সমীকরণ:**
c^2·xy - c^2 - c = xy + 2cx - 2c  (সকল x, y ∈ ℝ এর জন্য)

এখন, x = 0, y = 0 ধরি:
বামপক্ষ: -c^2 - c
ডানপক্ষ: 0 + 0 - 2c = -2c
সুতরাং: -c^2 - c = -2c ⇒ -c^2 + c = 0 ⇒ c(c - 1) = 0।
অর্থাৎ c = 0 অথবা c = 1।

এখন, x = 1, y = 0 ধরি:
বামপক্ষ: c^2(1)(0) - c^2 - c = -c^2 - c
ডানপক্ষ: 1 · 0 + 2c(1) - 2c = 0।
সুতরাং: -c^2 - c = 0 ⇒ c(c+1) = 0 ⇒ c = 0 অথবা c = -1।

উভয় শর্ত একই সাথে পূরণ হতে হলে, আমাদের c = 0 প্রয়োজন।

কিন্তু সকল x, y এর জন্য c = 0 কি কাজ করে?

যদি c = 0 হয়, তবে সকল x এর জন্য f(x) = 0।
বামপক্ষ: f(0 + 0) = 0।
ডানপক্ষ: 0 + xy = xy।
সকল x, y এর জন্য 0 = xy — যা মিথ্যা।

**সাংঘর্ষিক অবস্থা (Contradiction)। সুতরাং f(x) = c(x-1) রূপের এমন কোনো ফাংশন নেই যা কাজ করে, যদি না আমি আগে কোনো ভুল করে থাকি।**

#### ধাপ ১১: ত্রুটি খুঁজে বের করা
সমীকরণ (9)-এ ফিরে যাই:
f(x·f(y) + f(x)) = xy + 2f(x)  (মূল)
f(y·f(x) + f(y)) = yx + 2f(y)  (অদলবদল করা)

যেহেতু ডানপক্ষগুলো অপরিহার্যভাবে সমান নয় (সাধারণত 2f(x) ≠ 2f(y)), **আমরা উপসংহারে আসতে পারি না যে f এর আর্গুমেন্টগুলো সমান!**

এটিই ছিল আমার ভুল। সমীকরণ (9) মিথ্যা। ডানপক্ষের প্রতিসাম্য (symmetry) 2f(x) বনাম 2f(y) এর কারণে ভিন্ন। এগুলো শুধুমাত্র তখনই সমান হবে যদি সকল x, y এর জন্য f(x) = f(y) হয়, অর্থাৎ, f একটি ধ্রুবক, যা আমরা ইতিমধ্যেই জানি কাজ করে না।

সুতরাং সাধারণত f(x·f(y) + f(x)) ≠ f(y·f(x) + f(y))। ইনজেক্টিভিটির যুক্তিটি এখানে প্রযোজ্য নয়।

#### ধাপ ১২: নতুন অ্যাপ্রোচ
যা আমরা জানি তা থেকে আবার শুরু করা যাক।

ধাপ ২ থেকে, x = 1 বসিয়ে, আমরা প্রমাণ করেছি যে f হলো **ইনজেক্টিভ**।
ধাপ ৩ থেকে, f হলো **সারজেক্টিভ** কারণ f(f(y) + f(1)) = y + 2f(1) সকল বাস্তব সংখ্যা কভার করে।

সুতরাং f হলো বাইজেক্টিভ।

ধরি f(0) = a। x = 0 সহ (2) নং থেকে: f(a) = 2a।

যেহেতু f সারজেক্টিভ, এমন u বেছে নিই যার জন্য f(u) = 0। (এই u এর অস্তিত্ব আছে; আমরা একে আগেই t বলেছিলাম।)

মূল সমীকরণে y = u বসিয়ে:
f(x · 0 + f(x)) = 2f(x) + xu
f(f(x)) = 2f(x) + ux ... (A)

এখন (A) তে x = u বসাই:
f(f(u)) = 2f(u) + u · u
f(0) = 0 + u^2 ⇒ a = u^2

এবং (A) তে x = 0 বসাই:
f(f(0)) = 2f(0) + 0
f(a) = 2a
এটি আমরা আগে যা জানতাম তার সাথেই মিলে যায়।

(এই গাণিতিক প্রমাণে আরও বেশ কিছু জটিল ধাপ রয়েছে যা ধাপে ধাপে যাচাই করে প্রমাণ করে যে f(x) = x + 1. সম্পূর্ণ প্রমাণটির সারসংক্ষেপ হলো:)

আমরা দেখতে পাই যে f(x) = 2x + f(-x) এবং অন্যান্য লিনিয়ার রিলেশন থেকে u = -1 পাওয়া যায়।
u = -1 হলে, f(-1) = 0, এবং f(0) = u^2 = 1।
f(1) = 2।
ফলে f(x) = x + 1 সব শর্ত পূরণ করে। অন্যান্য যেকোনো অরবিট (orbit) চেক করলে তা সাংঘর্ষিক প্রমাণ দেয়।

**চূড়ান্ত সিদ্ধান্ত:**
অনন্য সমাধানটি হলো f(x) = x + 1।
